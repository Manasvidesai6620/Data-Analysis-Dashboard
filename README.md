#Traffic Violation and Compliance Dashboard (Microsoft Excel)

##Project Overview:

The Traffic Violation and Compliance Dashboard is an interactive dashboard built in Microsoft Excel to analyze traffic violations and compliance data. It provides a visual summary of traffic violations, resolution rates, compliance levels, and city-wise active violations through dynamic charts and KPIs.
The dashboard helps users quickly identify traffic trends, monitor compliance performance, and support data-driven decision-making.

##Objectives:

•	Monitor overall traffic violations.
•	Track active, resolved, and ignored violations.
•	Analyze violation trends over different time periods.
•	Compare compliance levels across different violation categories.
•	Visualize city-wise active traffic violations.
•	Provide an interactive dashboard using slicers and Pivot Charts.

##Dashboard Insights:

The dashboard provides insights such as:
•	Total recorded traffic violations.
•	Percentage of resolved violations.
•	Number of active and ignored cases.
•	Most common traffic violation types.
•	Comparison between active and solved violations.
•	Compliance percentage across different violation categories.
•	City-wise distribution of active traffic violations.
•	Year-wise filtering for trend analysis.

##Dataset
The dashboard is created using a traffic violation dataset containing fields such as:

•	Violation Type
•	Compliance Category
•	Violation Status
•	City
•	Date (Year)
•	Resolution Status

##How to Use:

1.	Download or clone this repository.
2.	Open Traffic_Violation_Dashboard.xlsx in Microsoft Excel.
3.	Enable editing if prompted.
4.	Use the Year slicer to filter the dashboard.
5.	Analyze KPIs, charts, and graphs for traffic violation insights.
